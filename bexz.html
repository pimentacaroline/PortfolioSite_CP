BEXZ

Bexz is a web app I created as a case study for my UX design course at CareerFoundry.To build it I performed tasks such as User Research, Information Architecture, and Interaction Design of the mobile version. The design challenge was to create a web-app (mobile-first approach) that “enables anyone, anywhere to instantly chat with an expert in virtually any field”.

My Process

Research

Based on my user research and analysis, I created user personas, user journey maps, task analysis, and user flows.

I created a script and conducted three in-person user interviews. The data were analyzed through Affinity Mapping, focussing on users' needs and pain points.

Ideation

I conducted a card sort test with OptimalSort and used the data to organize the sitemap content. Co-existing hierarchies will make it easy for users to access different pages.To avoid obstacles in the user experience, the app will allow users to try some features without signing up.

Wireframe

I started with paper wireframes to design the navigation. BEXZ is a browse-mostly app with more than five top-level navigations. Because of this, I chose a Top Navigation Bar for the main navigation and a Footer Navigation for the secondary pages.
